DiaQ is a Dialogue and Quest Engine with a graphical, node-based, editor component. This tool makes it easier to create the flow of conversations and to give the player quests and track the progress of those quests. Full C# source is provided, making it easy to modify the code if the need arise.

Note: This tool is included in UniRPG.
